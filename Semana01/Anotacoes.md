ALGORITMO
    CONJUNTO FINITO DE INSTRUCOES ORDENADAS PARA SOLUCIONAR UM PROBLEMA.




LOGICA DE PROGRAMACAO
    LOGICA -> 

    PROGRAMACAO -> PLANEJAR / ESCREVER INSTRUCOES-CODIGOS PARA ALGUM FIM (ROTEIRO) / AUTOMATIZAR-INFORMATIZAR / 

ESCREVER PROGRAMAS
    CODIGO FONTE
        TEXTO QUE ESCREVEMOS PARA DEFINIR O NOSSO ALGORITMO E MAIS TARDE SE TORNAR UM PROGRAMA

        LINGUAGEM DE PROGRAMACAO
            CONJUNTO DE REGRAS QUE DEFINEM UMA FORMA DE ESCREVER UM CODIGO FONTE

    COMPILACAO
        TRANSFORMA O CODIGO FONTE EM UM ARQUIVO DE INSTRUCOES PARA A MAQUINA.
        RESULTADO DO PROCESSO DE COMPILACAO É UM EXECUTAVEL
    
    EXECUTAVEL
        O QUE O USUARIO EXECUTA!
